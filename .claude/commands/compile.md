Compile all engram markdown configs into Hammerspoon Lua.

IMPORTANT: This runs non-interactively via `claude -p`. Do NOT ask questions or wait for confirmation. Just compile, validate, and report the results.

Use the engram-compile skill to compile configs from the `configs/` directory into Hammerspoon Lua modules at `~/.hammerspoon/engram/`.

This includes validation — compiled Lua is syntax-checked automatically.
